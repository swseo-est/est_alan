냉장고 구매 체크리스트

냉장고 구매 전 반드시 확인해야 할 핵심 체크리스트와 주요 고려사항을 확인하고 후회 없는 냉장고 선택을 하세요.

💡 TIP: 냉장고는 평균 10년 이상 사용하는 제품입니다. 기능성과 실용성을 모두 고려하세요.

📋 필수 체크사항

용량 계산하기

1인 가구: 250~300L / 2인 가구: 400~500L / 3~4인 가구: 600~700L / 5인 이상: 800L 이상

설치 공간 측정하기

설치 위치의 너비, 높이, 깊이와 통풍 공간 최소 5~10cm 확보 필요

에너지효율 등급 확인하기

1등급 제품이 2등급 대비 연간 약 2~3만원 전기요금 절약 가능

주요 기능 우선순위 정하기

UV항균, 홈바, 매직스페이스, AI 기능 등 실제 사용 빈도 고려

🔍 유형별 주요 고려사항

1인 가구 추천

소형 2도어(250~300L), 에너지효율 1등급 우선, 도어 알람 기능 권장, 성에 자동 제거 기능 유용

가족 가구 추천

3~4도어(600L+), 매직스페이스/홈바 유용, 도어인도어, 자동 얼음 메이커, 김치냉장고 별도 고려

공간 제약 시

키친핏/빌트인 모델, 코너 설치형, 가구장 틈새 4mm 제품, 도어 개폐 방향 확인 필수

비용 효율화

에너지 1등급 선택, 중국 제조 모델 고려(LG T/H 시리즈), 핵심 기능만 탑재된 모델 선택
